Enable AI Engine
📋 How to Enable Local AI (Step-by-Step)

Step 1 — Check your Chrome version

You need Chrome 127 or newer. Check yours at:

chrome://settings/help

Step 2 — Enable the Prompt API flag

Set it to Enabled:

chrome://flags/#prompt-api-for-gemini-nano

Step 3 — Enable On-Device Model

Set it to Enabled BypassPerfRequirement:

chrome://flags/#optimization-guide-on-device-model

Step 4 — Restart Chrome

Close all Chrome windows completely (use Cmd+Q on Mac) and reopen the browser. Come back to this page.

Step 5 — Wake Up the AI API

Click this button to force Chrome to recognize the new flags:

Step 6 — Download the Model (~2 GB)

Find "Optimization Guide On Device Model":

chrome://components

Click "Check for update" and wait until it says "Up-to-date". (It might say "Downloading" for a few minutes).

Step 7 — Done!

Once downloaded, toggle the AI Engine off and back on here. It should say ✅ AI Ready.
